# ssl-style-guide

A collection of syntax definition files for the STARLIMS Scripting Language (SSL), including:

- **Tree-sitter grammar** - For parsing and syntax highlighting in editors that support tree-sitter
- **TextMate Language grammar** - For syntax highlighting in VS Code, Sublime Text, and other TextMate-compatible editors
- **YAML Schema files** - For validation and autocompletion support
